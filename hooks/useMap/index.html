Result #

参数说明类型
mapMap 对象Ref<Map>
set添加元素(key: any, value: any) => void
get获取元素(key: any) => MapItem
setAll生成一个新的 Map 对象(newMap: Iterable<[any, any]>) => void
remove移除元素(key: any) => void
reset重置为默认值() => void

Params #

参数说明类型默认值
initialValue可选项，传入默认的 Map 参数Iterable<[any, any]>-
